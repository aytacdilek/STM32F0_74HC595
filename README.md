STM32F0_74HC595
===============

Just a simple 8-bit serial to parallel shift register ([ST] M74HC595) example for exercise.

Compiler: GNU-ARM

IDE:  CooCox

Programming Language: Embedded C

Development Environment:  STM32F0DISCOVERY
